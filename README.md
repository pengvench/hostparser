# 🌐 Multi-DNS Hosts Parser

Автоматизированный инструмент для сетевой отладки и проверки маршрутизации через публичные Smart DNS сервисы.

## 🚀 Возможности
- **Multi-DNS запрос:** Скрипт опрашивает несколько DNS-провайдеров одновременно.
- **Индивидуальные конфиги:** Генерация отдельных `hosts` файлов для каждого DNS-сервиса.
- **Автономность:** Работает как единый `.bat` файл (Hybrid Batch + PowerShell).
- **Портативность:** Не требует установки и сторонних библиотек.

## 🛠 Как пользоваться
1. Разместите `run.bat` и `domainlist.txt` в одной папке.
2. Запустите `run.bat`.
3. Полученные файлы из папки можно использовать для анализа ответов различных DNS-серверов.
